This folder contains the data and scripts for Exp. 1B.

Exp1B_ImgData_testSorted_243ss.csv: "test" data from Exp. 1B
Exp1B_QData_243ss.csv: demographic, mood and other participant-level data for Exp. 1B
Exp1B_Data_Codes.txt: code file for ImgData and QData

Exp1B_LMMs.R: compute linear mixed models and generate plots
